Day 30 Practice Quiz

Data Remanence & Sanitization — Domain 2: Asset Security
📝 10 Questions 🎯 Pass Mark: 7/10 ⏱️ ~12 minutes 📊 Advanced

Q1. A security engineer explains to junior staff that when a file is "deleted" using the OS recycle bin and emptied, the data is NOT removed from the disk. What term BEST describes the residual data that persists on the drive after deletion?

Q2. An organization is retiring HDDs classified as SECRET. Per NIST SP 800-88, which sanitization level makes recovery "infeasible even with state-of-the-art laboratory techniques" and should be applied when media will leave organizational control?

Q3. An SSD used in a finance server must be sanitized before being transferred to the HR department. Traditional multi-pass overwrite tools report the drive as sanitized, but NIST 800-88 warns these tools may not reach all physical blocks on SSDs. What sanitization method is MOST reliable for SSDs?

Q4. A media disposal vendor suggests degaussing all retired storage media. The security team objects that degaussing will be ineffective for certain device types. Which media type is NOT sanitized by degaussing?

Q5. NIST SP 800-88 defines three sanitization categories. Which of the following CORRECTLY maps all three levels to their primary method?

Q6. A data center uses self-encrypting drives (SEDs) for all servers. When a drive must be retired, the security operations team performs "cryptographic erasure." What operation does cryptographic erasure actually perform?

Q7. An organization transfers used laptops to a local charity. The drives contain former employee data classified as Internal. Which NIST SP 800-88 sanitization method is minimally required before this transfer since media is leaving organizational control?

Q8. A forensic researcher uses a magnetic force microscope to analyze the track edges of an overwritten hard drive and recovers fragments of the original data. This technique exploits which concept directly?

Q9. An external auditor requests evidence that decommissioned servers containing PII were properly sanitized before physical disposal by a third-party vendor. Which document BEST satisfies this audit requirement?

Q10. NAND flash memory (SSDs, USB drives) presents a unique sanitization challenge compared to magnetic HDDs. Which property of NAND flash is the ROOT CAUSE of this challenge?
