Day 31 Practice Quiz

Data Loss Prevention — Domain 2: Asset Security
📝 10 Questions 🎯 Pass Mark: 7/10 ⏱️ ~12 minutes 📊 Intermediate

Q1. A CISO wants to prevent employees from emailing sensitive customer records to their personal Gmail accounts. Which DLP deployment type MOST directly addresses this threat vector?

Q2. A healthcare organization deploys Network DLP at its internet gateway. The SOC reports that the DLP system is not detecting any sensitive PHI in HTTPS web uploads even though users are known to use web-based file sharing. What is the MOST likely cause?

Q3. An organization's DLP system is generating hundreds of false positive alerts per day — employees' product serial numbers (formatted as 16-digit strings) are triggering the credit card PAN detection rule. Which tuning approach BEST reduces false positives while maintaining PAN detection?

Q4. A software company wants its DLP system to detect if proprietary source code files (which have unique content and no standard format) are being exfiltrated. Which content inspection technique is MOST appropriate?

Q5. A remote employee working offline (disconnected from VPN) copies a Confidential financial model to a personal USB drive. Which DLP type would DETECT and BLOCK this action, and why?

Q6. An organization is implementing DLP for the first time. The security manager recommends deploying immediately in block mode to provide maximum protection. A senior analyst objects. What is the CORRECT recommended deployment sequence?

Q7. A CASB is deployed to extend DLP policies to cloud SaaS applications. For discovering sensitive data that has ALREADY been stored in SharePoint Online (not just monitoring new uploads), which CASB mode is MOST appropriate?

Q8. A DLP analyst notices that an insured employee has been emailing small fragments of a confidential merger document daily — no single email contains enough content to trigger the DLP rule. What technique does this represent, and which DLP approach would BEST address it?

Q9. Which of the following is an accurate statement about the relationship between data classification and DLP effectiveness?

Q10. A DLP system detects a file matching a Confidential fingerprint being attached to an email from a finance employee to an external auditor. Rather than blocking the transfer, the DLP system requires the employee to provide a business justification, logs the response, and allows the email to proceed. What DLP response action is this?
